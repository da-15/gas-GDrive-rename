# gas-GDrive-rename

Google Drive 上のファイル・フォルダ名を、スプレッドシートから一括変換する Google Apps Script ツールです。

---

## 機能

- 対象フォルダのファイル・フォルダ一覧をスプレッドシートに取得
- 変更後の名前を入力して一括リネーム
- フォルダ URL またはフォルダ ID どちらでも指定可能

---

## セットアップ

### 1. ファイルをダウンロード

[ZIPをダウンロード](https://github.com/da-15/gas-GDrive-rename/archive/refs/heads/main.zip) して解凍してください。

### 2. スプレッドシートに Apps Script を追加

1. Google スプレッドシートを新規作成して開く
2. メニューの **拡張機能 → Apps Script** を開く
3. デフォルトで表示されているコード（`function myFunction() { ... }`）を削除する
4. プロジェクトに以下の 2 ファイルを追加し、それぞれの内容をコピーする
   - `config.gs`
   - `rename_file_folder.gs`
5. メニュー中程の **プロジェクトを保存**（フロッピーアイコン）をクリック

### 3. 初回の認証

スプレッドシートをリロードするとメニューに **「GDrive名前一括変換」** が追加されます。  
初回実行時は Google アカウントの認証が求められます。  
認証手順については [こちら](https://coconala.com/blogs/2609655/113661) を参照してください。

## 使い方

スプレッドシートのメニュー「**GDrive名前一括変換**」から操作します。

| メニュー項目 | 説明 |
|---|---|
| フォルダのパスを指定する | 対象フォルダの URL または ID を入力し、一覧を取得する |
| 名前を一括変換する | 「変更後の名前」列に入力した内容でリネームを実行する |
| 一覧をクリアする | シートの内容をリセットする |
| TIPS を表示 | 操作手順のヒントを表示する |

### 基本的な流れ

1. **「フォルダのパスを指定する」** を選択し、Google Drive のフォルダ URL または ID を入力する
2. 一覧が表示されたら、**「ファイル / フォルダ名（変更後）」** 列に新しい名前を入力する（空欄の行はスキップされます）
3. **「名前を一括変換する」** を選択して実行する
4. **「処理」** 列に「済」が表示されたら完了

---

## スプレッドシートの列構成

| 列 | 内容 |
|---|---|
| Dir | フォルダの場合に `d` が入ります |
| ID | ファイル / フォルダの ID |
| ファイル / フォルダ名（元） | 現在の名前 |
| ファイル / フォルダ名（変更後） | 変更後の名前（ここに入力します） |
| 処理 | 実行結果（済 / エラー） |
